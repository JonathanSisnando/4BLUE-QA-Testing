# Projeto 4BLUE - Qualidade de Software

Este repositório contém toda a documentação de QA e artefatos de teste para o projeto **4BLUE**, focando na garantia de qualidade das funcionalidades de Criar Conta e Login.

## Objetivo
Validar as funcionalidades desenvolvidas de acordo com o Processo de Teste de Software, garantindo que o produto final atinja a qualidade desejada para o usuário final.

## Estrutura do Repositório
- `/docs/plan`: Plano de Teste detalhado com estratégia e escopo.
- `/docs/scenarios`: Mapeamento de cenários e casos de teste.
- `/docs/reports`: Relatórios de execução e parecer final da qualidade.

##  Ferramentas Utilizadas
- **Gestão de Issues:** GitHub Issues
- **Documentação:** Microsoft Office 365
- **Ambiente:** Web CMS (https://qa-play-sim.lovable.app/)
